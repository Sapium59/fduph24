{% extends "solution.html" %} {% block puzzle-author %} Wonderepublic {% endblock %} {% block solution-body-md %} 题目中展示了一些按字典序排列的声母和韵母，仔细观察可以发现以下特点： - 每行逗号前后的声韵母数量一致，且只有6和10两种数量； - 每个分句中的声母与韵母数量一致； 因此可以猜测需要将每个分句中的声韵母进行组合，从数量为6的句子入手是一个比较好的切入点，可以拼出一些比较熟悉的词组，如第五句中的“牵牛星”与“河汉女”就是一个比较明显的例子。在拼出了一些词句后，我们可以意识到这些都是包含了叠词的诗句，而在给出的声韵母碎片中删去了叠词。在掌握了这一规律之后，可以很快解出所有的这些诗句。 用括号中的数字作index提取该分句中被删去叠词的拼音，逐句阅读可以得到**thirtieth sentai**，即可得到最终答案**GOGO SENTAI**。

{% endblock %} {% block author-notes-html %} 这道题的灵感来自于GPH2022中的Extreme Anagram，一开始的设想是将汉语中一些有独特拼音特征的句子作类似的处理，但是后来一是觉得这样的形式缺乏主题性，二是没有找到足够多的特殊句子，于是就暂时搁置了这个想法。
在我尝试了用另一个思路出这道题未果后，我注意到了答案中的GOGO，忽然想到也许可以将叠词诗句作为处理的对象，于是先依原版将拼音的所有字母按字典序排列，然而验证下来可解性极低。后来改成了声韵母为对象，算是达到了一个比较好的平衡。

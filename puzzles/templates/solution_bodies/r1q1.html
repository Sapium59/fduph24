{% endblock %} {% block author-notes-html %} 每年台风季各个自媒体都会不厌其烦地发台风是怎么命名的，在又一次看到台风命名的推送之后就有了这道题。出题的过程中发现很多音译的台风名字的本来的意思还是很有趣的，可惜因为答案的限制最后选取的台风大多都是中国或者港澳命名的名字，少了一点发现音译名字的本来意思的乐趣。不过作为第一题还好，实际上很多人通过爆破得到的，也是降低了爆破的难度。 {% endblock %}

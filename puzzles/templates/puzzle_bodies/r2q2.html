Set A

在岛上的一个山洞中，你发现了规模宏大的壁画。这些真的是题吗？

Set B

你走到下一片谜题前，看到有两组题中间画上了等号。地上又有一张纸条，你捡了起来：

“如果你发现在新的区域内，有在你猜测的规则下无解或超过2解的题，不要紧张。

很有可能，你猜测的规则在某些情况下是正确的，但并不能涵盖所有情况。

换言之，你猜测的规则是真实规则的特例。

这个区域前两组题的解完全相同，我为你用等号标注了出来。请接着探索真实的规则。

若你毫无头绪，也许标题能帮助你。”

Set C

“我理解了。”你带着新的知识走向下一个区域。这回，数字出现在了网格内部而非外部。你隐约感到，网格内数字与网格外数字间有内在联系。

Set D

远处蓝色的光芒吸引了你。走近一看，谜题中出现了一些星星，发出淡淡的蓝色荧光。

Set E

“星星的效果真神奇，但仅仅如此吗？”你又看到了熟悉的等号。

Set F

山洞逐渐变得狭窄，你意识到自己即将抵达旅途的终点。

Final

你在山洞的尽头发现了一个像是笼子的石制建筑，大门紧紧关着。

门上的三个谜题，比你之前见过的任何都要大。

谜题下方有一些算式，旁边还有一行字：黑白相消，余数累和。
